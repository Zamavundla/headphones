# Web Page Project

This is a web page project built with strong foundations, including a reset CSS styling, the use of variables, simple and generic CSS selectors, and a simple HTML structure. The project follows best practices for building a web page and aims to provide a great user experience.

## Features

- Responsive Design: The web page switches to a mobile version when the screen width is 480px or less.
- Beautiful Typography: The web page utilizes the "Source Sans Pro" font for a clean and modern look.
- Eye-catching Animations: Links have a hover/active effect with the color #FF6565, creating an interactive experience.
- Centered Content: The maximum width of the content is set to 1000px and is centered on the page, ensuring optimal readability.

## Technologies Used

- HTML
- CSS

## CSS Libraries Used

The project utilizes the following CSS libraries:

- [Normalize.css](https://necolas.github.io/normalize.css/): A modern alternative to CSS resets that normalizes styles across different browsers.
- [Animate.css](https://animate.style/): A library of cross-browser CSS animations to add eye-catching effects to elements.

Make sure to include these CSS libraries in your project by linking to their respective stylesheets.

## Getting Started

1. Clone the repository: `git clone https://github.com/your-username/web-page-project.git`
2. Open the project folder in your preferred code editor.
3. Customize the HTML and CSS files to suit your needs.
4. Add any additional components or functionality as required.
5. Preview the web page in your web browser to see the changes.
